User Scenarios

    Opening the Game:
    User opens app on an iPhone or Android device, and places phone in VR Headset
    When the main menu pops up, the user may look at a button to select Play, Options, or Quit
    Playing the Game:
    User is placed in a cart on the track
    Cart will start to move forward along the track at a designated speed
    The user can turn their head to see the surrounding layout as the cart continues to move with the player in it.
    Enemies will pop out of the environment to be targets for the user to shoot
    User will clap when targeting an enemy to shoot the enemy
    A projectile will leave the player, and if the projectile connects with an enemy, points will be added to the scoreboard and the enemy will disappear
    At the end of the level, the score will be shown on the screen with an option to play again, move to the next level, or quit back to the main menu
    Pause Menu:
    User can select from Resume, Options, or Quit to Menu
    Selecting Options Menu:
    Change volume (Brightness Control to come)
    Go back to main menu
