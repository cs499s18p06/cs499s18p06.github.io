    • Test Play Button
      • Takes user to game mode -- Begins game
    • Test Options Button
      • Takes user to Options menu
      • Allows use to adjust volume/brightness of game
      • Test Back Button
        • Takes user back to main menu
  • Pause Menu
    • Pause Function
      • Test that game pauses
    • Test Resume Button
      • Takes user back to game mode
    • Test Options Button
      • Same Functionality as Options Button on Main Menu
  • Scoring
    • Scores are displayed at the end of the level (See Scoring for Rules)
  • Outline Colors Change
    • Scoring
      • Black Outline = 5 points
      • Blue Outline = 10 points
      • Gold / Yellow Outline = 15 points
    • Red Outline + Monster Fill Color is Red = Monster is Hit

    Integration and Function Testing
  • Pause Menu
    • User is able to access menu within the game
  • Combat
    • Firing when user presses button on VR headset
    • Monster turns red on hit
    • Hit Criteria
      • Reticle aimed at monster (within colored outline)
      • Button Pressed (firing)
    • Miss Criteria
      • Reticle aimed at / outside monster outline
  • Terrain Elements
    • Tombstones and Trees change based on microphone input
      • Size of tombstones
      • Size of leaves
    • Art style is consistent throughout the level
      • Terrain Objects (buildings, tunnels, etc) are not the default white color
    • Objects are appropriately placed
      • There aren't floating tombstones (or other stationary elements)
  • Game Operates Smoothly
    • Track movement does not glitch
    • Monsters appear with sufficient time to hit them (3 seconds)
    • Firing is accurate
    • Firing occurs without lapse between button press and target hit
    • Camera angle is first person
  • Scoring
    • Scores are displayed at the end of the level
    • Monsters are outlined in different colors depending on scoring criteria
      • Black Outline = 5 points
      • Blue Outline = 10 points
      • Gold / Yellow Outline = 15 points

    System Testing
  • Compatibility Testing
    • Test iOS Compatibility
      • iPhone 6s Running iOS version 11.3
    • Test Android Compatibility
      • Google Pixel 2 XL Running OS version 8.1

    Customer Testing
  • Monster is hit and turns red (fill and outline color)
